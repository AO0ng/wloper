# 水文数据自动化抓取工具 - 项目总结

## 一、项目概述

本项目是一个基于 **Node.js + Playwright** 的水文数据自动化抓取工具，主要用于从江西省水文系统中自动抓取站点的逐日平均流量数据，并生成周总量统计报告。

## 二、项目结构

```
work/
├── scraper.js          # 主爬虫脚本 - 核心功能实现
├── inspect.js          # 网站检查脚本 - 分析登录流程
├── explore_new.js      # HTML分析脚本 - 探索新站点结构
├── chk.js              # HTML检查脚本 - 检查可点击元素
├── debug_store.js      # 调试数据存储
├── debug_fields.js     # 调试日志记录
├── package.json        # 项目配置（实际包含API测试代码）
├── package-lock.json   # 依赖锁定文件
├── run.ps1             # PowerShell运行脚本
├── setup.bat           # Windows安装脚本
├── page.html           # 测试页面
├── debug/              # 错误截图目录
├── debug_inspect/      # 调试截图和HTML目录
└── logs/               # 运行日志目录
```
